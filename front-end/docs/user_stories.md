User Stories 
------------

```
As a user,
So that I can choose a song,
I would like the tracks to be displayed. 

As a user,
So that I can enjoy music,
I want to be able to play back songs.

As a band,
So that we can attract more fans,
I would like to be able to upload a song.

As a band,
So that I can fund my music making,
I would like to be able to receive payments.

As a user,
So that i can support the artists whose music I enjoy,
I would like to be able to pay money to the artist.

As a band,
So that I can see how successful we are,
I would like to be able to see how many times a track has been played.

As a music lover,
So that I can support artists and get access to more features,
I would like to be able to subscribe to the service.


As a music lover,
So that I can view additional features,
I would like to be able to sign in.

As a user,
So that I can find the music I am interested in,
I would like to be able to search tracks.

```
